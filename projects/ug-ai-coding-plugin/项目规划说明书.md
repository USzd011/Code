## 一、项目概述

### 1.1 项目定位

打造一款运行在 **Siemens NX 12.0** 中的 AI 自动编程插件，实现从 **零件图纸/3D模型 → 工艺规划 → G代码生成 → 刀路仿真 → 机床验证** 的全流程自动化。

**开发策略：**
- ✅ **第一阶段**：全力开发 UG 12.0 插件
- ✅ **第二阶段**：适配 MC (MACH) 插件
- ✅ **第三阶段**：兼容其他 NX 版本（12.0 ~ 1980+）

### 1.2 核心价值

- **效率提升**：传统编程 2-4 小时 → AI 自动编程 5-10 分钟
- **降低门槛**：初级编程员也能输出专业级工艺
- **质量保障**：刀路仿真验证，碰撞/过切提前发现
- **知识沉淀**：工艺参数库持续积累，越用越聪明

### 1.3 目标用户

- 数控编程工程师（主力用户）
- 工艺工程师
- 车间班组长

## 二、功能架构

```

│                    用户交互层                          │
```

## 五、数据库设计

### 5.1 现有表结构

```sql
-- 用户管理
CREATE TABLE users (
    id INTEGER PRIMARY KEY,
    username TEXT UNIQUE,
    password_hash TEXT,
    role TEXT DEFAULT 'operator',
    created_at TIMESTAMP
);

-- 操作日志
CREATE TABLE operation_logs (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    action TEXT,
    details TEXT,
    created_at TIMESTAMP
);

-- 自定义材料
CREATE TABLE custom_materials (
    id INTEGER PRIMARY KEY,
    name TEXT,
    density REAL,
    hardness REAL,
    thermal_conductivity REAL,
    is_standard INTEGER DEFAULT 0
);

-- 工艺方案
CREATE TABLE process_plans (
    id INTEGER PRIMARY KEY,
    part_name TEXT,
    material TEXT,
    tool_list TEXT,  -- JSON
    parameters TEXT, -- JSON
    gcode TEXT,
    status TEXT,
    created_at TIMESTAMP
);

-- 刀具库
CREATE TABLE tool_library (
    id INTEGER PRIMARY KEY,
    name TEXT,
    type TEXT,       -- 面铣/立铣/球头/车刀/钻头/镗刀
    material TEXT,   -- 硬质合金/高速钢/陶瓷/立方氮化硼
    coating TEXT,    -- TiN/TiAlN/AlCrN/DLC
    diameter REAL,
    flute_length REAL,
    total_length REAL,
    max_rpm INTEGER,
    max_feed REAL
);

-- 加工记录
CREATE TABLE machining_records (
    id INTEGER PRIMARY KEY,
    plan_id INTEGER,
    actual_time REAL,
    actual_cost REAL,
    quality_score REAL,
    issues TEXT,
    created_at TIMESTAMP
);
```
