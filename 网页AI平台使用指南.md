# 网页AI平台使用指南

## 🎯 目标
通过浏览器手动操作网页AI平台，获取高质量回答。

---

## 📋 平台列表

### 1. LMArena（推荐）
- **地址**: https://lmarena.ai
- **特点**: 多模型对比测评
- **使用方法**:
  1. 打开网站
  2. 输入问题
  3. 查看多个模型的对比回答
  4. 选择最佳答案

### 2. XSimple Chat
- **地址**: https://xsimplechat.com/chat
- **特点**: 聚合多模型对话
- **使用方法**:
  1. 打开网站
  2. 选择模型
  3. 输入问题
  4. 获取回答

### 3. 蓝镜ChatGPT中文版
- **地址**: https://ai.lanjingchat.com
- **特点**: 中文友好
- **使用方法**:
  1. 打开网站
  2. 输入问题
  3. 获取回答

### 4. Gemini 官网
- **地址**: https://gemini.google.com
- **特点**: Gemini对话界面
- **使用方法**:
  1. 打开网站
  2. 登录Google账号
  3. 输入问题
  4. 获取回答

---

## 💡 使用建议

### 追求最佳结果
1. **使用LMArena对比多个模型**
2. **选择回答最好的模型**
3. **复制结果到项目中**

### 快速对话
1. **使用XSimple Chat或蓝镜ChatGPT**
2. **无需登录，直接使用**

### 中文任务
1. **优先使用蓝镜ChatGPT**
2. **中文理解和表达更好**

## 📊 效率提升

### 批量处理
1. 准备问题列表
2. 逐个输入平台
3. 收集所有回答
4. 整理最佳答案

### 模板使用
```
问题模板：
- 请解释[概念]

- 请设计[系统]的架构
```

## ⚠️ 注意事项

1. **网络要求**: 部分平台需要科学上网
2. **频率限制**: 避免过度调用
3. **结果验证**: 检查回答的准确性
4. **隐私保护**: 不要输入敏感信息
