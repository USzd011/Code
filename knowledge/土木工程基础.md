### 1.2 内力分析

#### 轴力（N）
- **拉杆**: N > 0（拉力）
- **压杆**: N < 0（压力）
- **计算公式**: N = P/A
- **应力**: σ = N/A

#### 剪力（V）
- **截面两侧相对滑动趋势**
- **计算公式**: V = ΣP
- **剪应力分布**: 矩形截面呈抛物线，圆形截面呈椭圆

#### 弯矩（M）
- **截面一侧受拉，一侧受压**
- **计算公式**: M = Σ(P × d)
- **正负规定**: 使梁下侧受拉为正
- **弯曲应力**: σ = My/I

#### 扭矩（T）
- **截面绕轴线旋转**
- **计算公式**: T = P × r
- **扭转角**: φ = TL/(GJ)
- **剪应力**: τ = Tr/J

```
σx, σy, τxy

主应力:
σ₁,₂ = (σx+σy)/2 ± √[((σx-σy)/2)² + τxy²]

主方向:
tan2θ = 2τxy/(σx-σy)

最大剪应力:
τmax = (σ₁-σ₂)/2
```
