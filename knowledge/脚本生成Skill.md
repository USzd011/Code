# 脚本生成Skill - video-script-writer

> 来源：小红书 - 瑶瑶AI增长
> 整理时间：2026-05-04

---

## 一、Skill概述

**名称：** video-script-writer
**功能：** 写脚本
**触发词：** "帮我写口播稿"

---

## 二、核心功能

### 2.1 主要任务

**基于角度卡片生成瑶瑶风格口播稿：**
```
纯口语、像朋友聊天、有立场、多案例
```

### 2.2 输出内容

**输出：**
```
口播稿 + 脚本标注 + 时长预估 + 画面提示
```

---

## 三、脚本结构

### 3.1 五段结构

**完整结构：**
```
钩子 → 降维 → 案例×3 → 立场 → 行动
```

### 3.2 各段说明

**第1段：钩子**
```
前3秒抓住观众注意力
```

**第2段：降维**
```
降低认知门槛，让观众容易理解
```

**第3段：案例×3**
```
3个具体案例，增强说服力
```

**第4段：立场**
```
表达明确立场，形成观点
```

**第5段：行动**
```
引导观众行动
```

---

## 四、时长要求

### 4.1 时长范围

**时长：**
```
60-90秒
```

### 4.2 字数范围

**字数：**
```
300-450字
```

---

## 五、瑶瑶风格特点

### 5.1 四大特点

**特点1：纯口语**
```
不书面化，像日常说话
```

**特点2：像朋友聊天**
```
亲切自然，不生硬
```

**特点3：有立场**
```
观点明确，不模棱两可
```

**特点4：多案例**
```
具体案例增强说服力
```

---

## 六、脚本生成流程

### 6.1 完整流程

```
读取角度卡片 → 分析角度 → 生成口播稿 → 添加标注 → 估算时长 → 添加画面提示
```

### 6.2 关键步骤

**步骤1：读取角度卡片**
```
理解选题角度
明确核心观点
```

**步骤2：分析角度**
```
确定内容方向
设计论证逻辑
```

**步骤3：生成口播稿**
```
按五段结构生成
保持瑶瑶风格
```

**步骤4：添加标注**
```
标注关键点
标注情绪变化
```

**步骤5：估算时长**
```
控制在60-90秒
字数300-450字
```

**步骤6：添加画面提示**
```
标注画面配合
提示演示时机
```
