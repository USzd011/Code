### 案例1：法兰盘加工（从零开始）

#### 零件图纸分析
```
零件名称：法兰盘
材料：45钢，调质处理，硬度HB220-250
批量：小批量，单件生产
特点：外圆+端面+螺纹+键槽+圆弧
难点：位置精度高，需保证同轴度和垂直度
```

```gcode
G02 X45.0 Z-50.0 C180.0 F50

; ... 继续铣削，直到完成整个圆弧轮廓

; ========== 取消C轴旋转 ==========
; C轴归零，回到初始位置

M30            ; 程序结束
```

### 案例2：曲轴加工（高级应用）

#### 零件特点
- **材料**：40Cr锻钢，正火处理
- **特点**：偏心轴，多个连杆轴颈，需车削+铣削+磨削
- **难点**：偏心距大，平衡要求高，加工难度大
- **关键尺寸**：
  - 主轴颈直径：φ50mm
  - 连杆轴颈直径：φ45mm
  - 偏心距：25mm
  - 连杆轴颈数：4个，间隔90度

#### 工艺规划
| 工序 | 内容 | 刀具 | 设备 | 备注 |
|------|------|------|------|------|
| 1 | 坯料准备 | - | - | 锻造，正火 |
| 2 | 车两端，钻中心孔 | T0101 | 车刀/钻头 | 保证同轴度 |
| 3 | 车偏心部分 | T0101 | 车刀 | 保证偏心距 |
| 4 | 铣削连杆轴颈 | T0404 | 铣刀（动力刀） | 保证角度 |
| 5 | 车削平衡块 | T0101 | 车刀 | 保证动平衡 |
| 6 | 磨削加工 | - | 磨床 | 高精度 |
| 7 | 探伤检测 | - | - | 无损检测 |

#### 编程要点

##### 步骤1：计算偏心距
```
偏心距 E = (D/2) - sqrt((D/2)^2 - (L/2)^2)
其中：
- D: 曲柄半径（连杆轴颈中心到主轴中心的距离）
- L: 连杆长度

示例：
- 曲柄半径 D = 50mm
- 连杆长度 L = 100mm
- 偏心距 E = 25mm
```

##### 步骤2：C轴分度编程
```gcode
O0004          ; 曲轴-多工位加工程序
G90 G54 G94

; ========== 车削偏心部分 ==========
T0101 M03 S2000
G00 X50.0 Z2.0

; 车削主轴颈
Z-50.0
X49.98 F50    ; 精车，直径49.98mm
; ... 继续车削

; 车削偏心部分（使用G51.1/G50.1镜像）
; 方法1：使用镜像指令
G51.1 P_      ; 镜像开始，P为镜像轴号
G00 X_ Z_
; ... 加工偏心部分
G50.1        ; 镜像结束

; 方法2：使用C轴分度（推荐）
; 对于每个连杆轴颈，C轴旋转对应角度
FOR i = 0 TO 3
    ; C轴旋转到第i个工位
    C_angle = i * 90.0  ; 4个工位，间隔90度
    G01 C(C_angle) F10.0
    
    ; 铣削当前工位的连杆轴颈
    ; 使用动力刀铣削
    G01 X_ Z_ C_ F_
NEXT i
```

##### 步骤3：完整程序框架
```gcode
O0004          ; 曲轴加工程序
G90 G54 G94

; ========== 初始化 ==========
T0101 M03 S2000
G00 X50.0 Z2.0

; ========== 车削主轴颈 ==========
Z-50.0
X49.98 F50
; ... 继续车削主轴颈

; ========== 车削偏心部分 ==========
; 使用C轴分度，依次加工4个连杆轴颈
FOR i = 0 TO 3
    ; C轴旋转到第i个工位
    C_angle = i * 90.0
    G01 C(C_angle) F10.0
    
    ; 车削连杆轴颈
    ; 根据偏心距计算X坐标
    ; X = 主轴中心 + 偏心距 * cos(角度)
    ; Z根据具体尺寸确定
    
    ; 示例：第一个连杆轴颈
    ; X = 25.0 + 25.0 * cos(0°) = 50.0mm
    ; Z = -30.0mm
    
    G01 X50.0 Z-30.0 F50
    ; ... 继续车削
NEXT i

; ========== 铣削连杆轴颈 ==========
; 换动力刀
T0404 M06
G00 X50.0 Z-30.0
G43 Z-25.0 H04
M03 S3000

FOR i = 0 TO 3
    ; C轴旋转到第i个工位
    C_angle = i * 90.0
    G01 C(C_angle) F10.0
    
    ; 铣削连杆轴颈轮廓
    ; 使用G02/G03圆弧插补
    G02 X_ Z_ C_ F_
    ; ... 铣削轮廓
NEXT i

; ========== 车削平衡块 ==========
; 在适当位置车削平衡块，保证动平衡
; 使用G51.1/G50.1镜像
G51.1 P_
; ... 车削平衡块
G50.1

; ========== 程序结束 ==========
M05
M30
```

## 三、常见问题与解决

### 问题1：C轴定位不准

**现象：** C轴旋转后，实际位置与指令不符，误差较大。

**原因分析：**
- 编码器反馈异常
- 机械传动间隙
- 参数设置不当
- 回零不准确

**解决方法：**
```gcode
; 1. 重新回零，校准参考点
G28 U0 W0    ; 回X轴参考点
G28 U0 Z0    ; 回Z轴参考点

; 2. 检查编码器连接
;    - 确保电缆连接牢固
;    - 检查屏蔽层接地
;    - 更换损坏的编码器线缆

; 3. 调整C轴参数
# 增大定位精度参数
# N202 = 0.005  ; 提高定位精度

; 4. 使用原点回归功能
G112      ; 取消C轴锁定
G113      ; C轴回原点

; 5. 机械调整
;    - 检查联轴器是否松动
;    - 调整传动间隙
;    - 润滑传动部件
```

### 问题2：动力刀抖动

**现象：** 动力刀旋转时振动大，噪音异常。

**原因分析：**
- 刀具不平衡
- 刀具安装不牢
- 主轴跳动过大
- 切削参数不当

**解决方法：**
```gcode
; 1. 检查刀具平衡
;    - 使用动平衡仪检测
;    - 进行静平衡校正
;    - 更换不平衡刀具

; 2. 检查刀具安装
;    - 确保刀具安装到位
;    - 使用合适的刀柄
;    - 紧固力矩符合要求

; 3. 检查主轴状态
;    - 测量主轴跳动
;    - 修复或更换主轴轴承
;    - 清洁主轴锥孔

; 4. 调整切削参数
;    - 降低转速
;    - 减小进给
;    - 选择合适的切削深度
```

### 问题3：加工尺寸超差

**现象：** 加工后的零件尺寸不符合图纸要求。

**原因分析：**
- 刀具磨损
- 热变形
- 装夹变形
- 测量误差

**解决方法：**
```gcode
; 1. 及时换刀
;    - 定期测量刀具磨损
;    - 达到磨损极限及时更换
;    - 记录刀具寿命数据

; 2. 补偿热变形
;    - 测量温度变化
;    - 计算热膨胀量
;    - 修改程序补偿

; 3. 改进装夹
;    - 使用软爪，减少变形
;    - 增加支撑，提高刚性
;    - 均匀夹紧，避免变形

; 4. 精确测量
;    - 使用精密量具
;    - 多次测量取平均值
;    - 修正测量误差
```

### 问题4：表面质量差

**现象：** 加工表面粗糙，有振纹、划痕等。

**原因分析：**
- 切削参数不当
- 刀具刃口钝化
- 振动干扰
- 冷却不足

**解决方法：**
```gcode
; 1. 优化切削参数
;    - 提高转速，降低进给
;    - 减小切深，增加走刀次数
;    - 选择合适的切削速度

; 2. 保持刀具锋利
;    - 定期修磨刀具
;    - 使用涂层刀具
;    - 及时更换磨损刀具

; 3. 减少振动
;    - 增加支撑，提高刚性
;    - 调整切削参数，避开共振频率
;    - 使用减振装置

; 4. 改善冷却
;    - 充分冷却，降低温度
;    - 使用切削油，减少摩擦
;    - 合理设计冷却液流向
```

### 4.3 安全注意事项

#### 操作前检查
- [ ] 确认机床各部位完好，无异常声响
- [ ] 确认防护装置齐全，门锁正常
- [ ] 确认急停按钮有效，可随时按下
- [ ] 确认冷却液充足，润滑良好
- [ ] 确认工件装夹牢固，刀具安装正确
- [ ] 确认周围无人员，通道畅通

#### 操作中注意
- [ ] 严禁戴手套操作，长发必须束起
- [ ] 严禁在运行中触摸工件、刀具
- [ ] 严禁跨越运转中的部件
- [ ] 严禁擅自修改程序参数
- [ ] 发现异常立即停机，报告上级
- [ ] 不得离开工作岗位，如需离开须关机

#### 操作后处理
- [ ] 清理机床，擦拭干净
- [ ] 填写交接班记录，注明异常情况
- [ ] 关闭总电源，锁好门窗
- [ ] 做好设备点检，发现问题及时上报

## 五、进阶学习路径

### 5.1 初级阶段（1-3个月）
- 掌握基础G代码编程
- 熟悉新代系统操作界面
- 完成简单零件加工
- 编写标准程序
- 积累实际加工经验

### 5.2 中级阶段（3-6个月）
- 学习宏程序开发
- 掌握车铣复合编程
- 熟练使用CAM软件（Mastercam/UG）
- 具备工艺分析能力
- 能够解决一般问题

### 5.3 高级阶段（6-12个月）
- 研究先进制造技术
- 掌握多轴加工编程
- 具备系统集成能力
- 能够解决复杂工程问题
- 参与项目研发

### 5.4 专家阶段（1-3年+）
- 成为某一领域的专家
- 推动技术创新和成果转化
- 带领团队完成复杂项目
- 在行业内建立影响力
- 培养新人，传承经验
