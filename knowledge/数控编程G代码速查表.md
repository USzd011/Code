|-------|----------|------|------|

| **G09** | 准确停止 | G09 G01 X100 Y50 | 到达目标点后精确停止 |
| **G17-G19** | 平面选择 | G17(XY), G18(XZ), G19(YZ) | 选择加工平面 |
| **G20-G21** | 单位制 | G20(英制), G21(公制) | 设定尺寸单位 |

| **G29** | 从参考点返回 | G29 | 从G28设置的点返回 |

| **G43-G49** | 长度补偿 | G43 H1 Z50 | 刀具长度补偿 |

| **G70-G72** | 精加工循环 | G71(外径粗车) | 固定循环粗加工 |
| **G80-G89** | 孔加工循环 | G81(钻孔) | 孔加工固定循环 |

## 五、典型编程示例

### 5.1 外圆车削
```gcode
O0001
T0101(外圆刀)
G00 X50 Z2 M03 S800
G42 G01 X35 F0.2(倒角)
Z-20 F0.1(外圆)
X40(退刀)
Z2(回起刀点)
G40 X50(取消刀补)
M30
```

### 5.2 内孔镗削
```gcode
O0002
T0202(内孔刀)
G00 X20 Z2 M03 S600
G41 G01 X18 Z-15 F0.1(内孔)
Z-30(深孔)
X22(直径)
Z2(回起刀点)
G40 X20(取消刀补)
M30
```

### 5.3 螺纹加工
```gcode
O0003
T0303(螺纹刀)
G00 X28 Z3 M03 S300
G92 X27.5 Z-20 F1.5(第一刀)
X27.1(第二刀)
X26.8(第三刀)
X26.5(第四刀)
G00 X50 Z50(退刀)
M30
```

### 5.4 圆弧插补
```gcode
O0004
G01 X10 Y10(起点)
G02 X30 Y10 I10 J0(顺时针R20)
G03 X30 Y30 I0 J10(逆时针)
G01 X10 Y30(回到起点)
M30
```

### 5.5 固定循环(钻孔)
```gcode
O0005
G90 G54(绝对编程)
G00 X10 Y10(定位)
G43 Z5 H1(长度补偿)
G99 G81 X20 Y20 Z-10 R2 F50(钻孔循环)
Y30(第二个孔)
G80(取消循环)
G00 Z100(抬刀)
M30
```

## 六、常见错误及解决方法

| 错误现象 | 可能原因 | 解决方法 |
|----------|----------|----------|
| 刀具不按路径移动 | G代码格式错误 | 检查G代码格式，确保正确 |
| 尺寸偏差大 | 进给速度过快 | 降低进给速度，提高精度 |
| 圆弧不光滑 | G02/G03使用不当 | 检查圆心偏移量(I/J/K) |
| 撞刀事故 | 刀补计算错误 | 重新计算刀补，验证路径 |
| 程序不执行 | M代码缺失 | 添加M03(主轴正转)等M代码 |
| 加工表面粗糙 | 刀具磨损 | 更换新刀具，调整参数 |
| 尺寸超差 | 测量误差 | 使用精密量具，多次测量 |
| 程序乱码 | 编码格式错误 | 统一编码格式(GBK/UTF-8) |
| 系统报警 | 参数设置错误 | 检查系统参数，恢复默认值 |
| 加工不稳定 | 夹具松动 | 紧固夹具，检查装夹 |
| 主轴过热 | 冷却不足 | 增加冷却液流量，检查泵 |
| 刀具寿命短 | 切削参数不当 | 优化切削参数，选择合适的刀具 |
| 振动噪音大 | 刚性不足 | 增加支撑，减小悬伸 |
