## 一、核心理念

**OpenClaw的重点：**
> 不是"能不能聊天"，而是能不能接入你每天真正用的工具

**核心价值：**
- 笔记、任务、邮件、日历、代码、语音、PDF、智能家居都能被串起来
- 不只是AI聊天框，而是一个**自动化工作台**

## 二、完整技能清单（30个）

### 2.1 笔记类（3个）

| 序号 | 技能名 | 功能描述 |
|------|--------|----------|
| 1 | **obsidian** | 管理本地Obsidian笔记库，创建、编辑、搜索笔记 |
| 2 | **notion** | 集成云端Notion，管理笔记、数据库、页面 |

## 五、安装指南

### 5.1 通过clawhub安装

```bash
# 安装单个技能
openclaw skill install <skill-name>

# 批量安装
openclaw skill install obsidian notion github

# 查看已安装技能
openclaw skill list

# 更新技能
openclaw skill update <skill-name>
```

### 5.2 推荐安装顺序

**第1步：系统基础**
```
clawhub + healthcheck
```

**第2步：核心工具**
```
summarize + nano-pdf + weather
```

**第3步：按场景安装**
```
根据使用场景选择对应技能组合
```
