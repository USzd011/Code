## 一、数控机床概述

### 1.1 什么是数控机床

**定义：**
数控机床（CNC机床）是采用数字化信号控制的机床，通过程序控制刀具和工件的相对运动，实现复杂零件的加工。

**核心优势：**
- ✅ 高精度：重复定位精度可达±0.001mm
- ✅ 高效率：自动换刀、自动加工
- ✅ 高柔性：只需修改程序即可加工不同零件
- ✅ 自动化：可实现无人化生产

### 1.2 数控机床分类

| 类型 | 特点 | 应用场景 |
|------|------|----------|
| **数控车床（CT）** | 回转体加工 | 轴类、盘类零件 |
| **数控铣床（MT）** | 平面/轮廓加工 | 模板、模具、箱体 |
| **加工中心（MC）** | 带刀库自动换刀 | 复杂零件批量加工 |
| **数控磨床（GT）** | 精密磨削 | 高精度表面加工 |
| **数控线切割（WEDM）** | 放电切割 | 异形孔、复杂轮廓 |
| **数控冲床（PT）** | 板材冲压 | 钣金件加工 |

## 三、G代码深度解析

### 3.1 运动控制G代码

#### 3.1.1 定位指令
```gcode
G00    - 快速定位（点位控制）
       - 路径：最短路径，不切削
       - 速度：各轴独立快移，非联动
       - 应用：空行程移动

G01    - 直线插补（轮廓控制）
       - 路径：直线，按F进给
       - 速度：联动，恒定进给
       - 应用：外圆、内孔、端面

G02/G03 - 圆弧插补
       - G02: 顺时针圆弧
       - G03: 逆时针圆弧
       - 需指定终点或半径
       - 应用：圆弧轮廓
```

#### 3.1.2 圆弧插补详解
```gcode
; 绝对方式编程
G02 X__ Y__ I__ K__ F__
; I: 圆心相对于起点的X增量
; K: 圆心相对于起点的Z增量

; 相对方式编程
G02 X__ Y__ I__ K__ F__

; 示例：顺时针圆弧
G02 X50.0 Y30.0 I25.0 K0.0 F100
; 圆心在(25,0)，起点(0,0)，终点(50,30)

; 示例：逆时针圆弧
G03 X30.0 Y50.0 I-10.0 J15.0 F80
; 圆心在(-10,15)
```

### 3.2 坐标系G代码

#### 3.2.1 坐标系设定
```gcode
G54-G59   - 工件坐标系（最多6个）
           - 用于多工件加工
           - 通过MDI设定偏移量

G90/G91   - 绝对/相对编程
           - G90: 所有坐标为绝对值
           - G91: 所有坐标为增量值

G92       - 设定工件坐标系原点
           - 格式：G92 X_ Y_ Z_
           - 应用：对刀后设定零点

; 示例：设定工件坐标系
G92 X0 Y0 Z0     ; 设当前点为零点
G54              ; 选择G54坐标系
```

#### 3.2.2 平面选择
```gcode
G17 - XY平面（默认）- 适用于铣床
G18 - ZX平面 - 适用于车床
G19 - YZ平面 - 特殊加工
```

### 3.3 刀具补偿G代码

#### 3.3.1 半径补偿
```gcode
G40 - 取消刀具半径补偿
G41 - 刀具左补偿（沿前进方向看，刀具在左侧）
G42 - 刀具右补偿（沿前进方向看，刀具在右侧）

; 使用要点：
; 1. 补偿必须在G01/G00模式下进行
; 2. 补偿过程不能中断
; 3. 补偿结束后必须用G40取消

; 示例：外圆粗加工
G42 G01 X50.0 Y10.0 F100 D01  ; 右补偿，刀具号01，刀补号01
...                    ; 加工外圆
G40 G00 X0 Y0             ; 取消补偿，退回
```

#### 3.3.2 长度补偿
```gcode
H/D地址 + 补偿号
; H: 西门子系统
; D: 新代等系统

; 示例：钻头磨损补偿
G00 Z100.0
T0101 M06            ; 换1号刀，刀补1
G43 Z50.0 H01        ; 长度补偿，刀补号01
G99 G81 X20.0 Z-30.0 R3.0 F80  ; 钻孔循环
...
G80                  ; 取消循环
M05                  ; 主轴停止
```

### 3.4 固定循环G代码

#### 3.4.1 钻孔循环
```gcode
G81 - 普通钻孔循环
G82 - 钻孔+暂停（锪平底孔）
G83 - 深孔啄钻（断屑）
G84 - 攻螺纹循环
G85 - 镗孔循环（正向）
G86 - 镗孔循环（单向）
G87 - 反镗循环
G88 - 镗孔循环（手动）
G89 - 镗孔循环（暂停）

; G81示例
G90 G54 G00 X20.0 Y10.0 S3000 M03
G43 Z5.0 H01
G81 X20.0 Y10.0 Z-25.0 R3.0 F50
Y30.0                ; 第二个孔
X50.0
G80                  ; 取消循环
M05
M30
```

#### 3.4.2 攻螺纹循环
```gcode
G84 - 刚性攻丝（主轴正转+反转）
G74 - 反向攻丝（主轴反转+正转）

; 参数设置
S: 主轴转速（根据材料调整）
F: 螺距（导程）
R: 参考平面高度

; 示例
G84 X20.0 Y10.0 Z-20.0 R3.0 F1.5
; M1.5螺纹，转速约1500rpm
```

### 3.5 辅助功能M代码

#### 3.5.1 主轴控制
```gcode
M03 - 主轴正转（顺时针）
M04 - 主轴反转（逆时针）
M05 - 主轴停止
M19 - 主轴定向停止
Sxxxx - 主轴转速（rpm）
M08 - 冷却液开
M09 - 冷却液关
```

#### 3.5.2 换刀与刀库
```gcode

; T0101: 选1号刀，换1号刀，刀补1
; T0202: 选2号刀，换2号刀，刀补2
```

## 六、动力刀塔编程

### 6.1 动力刀塔概述

#### 基本概念
- **定义**：车床上安装的动力刀具系统，可进行铣削、钻孔等加工
- **优势**：实现车铣复合，减少装夹次数，提高精度
- **应用**：曲轴、凸轮轴、齿轮箱等复杂零件

#### 主要厂商
- **新代（SYNTECH）**：国内主流，性价比高
- **三菱（MITSUBISHI）**：日系，精度高
- **马扎克（MAZAK）**：高端，性能卓越
- **德玛吉（DMG MORI）**：德国，顶级配置

### 6.2 C轴控制

#### C轴基本功能
```gcode
; C轴分度
G01 C_angle F_    ; 旋转指定角度
; 示例：分度90度
G01 C90.0 F10.0

; C轴联动（螺旋插补）
G02/G03 X_ Z_ C_  ; 螺旋插补
; 示例：螺旋槽加工
G02 X20.0 Z-10.0 C90.0 F50
```

#### C轴编程示例
```gcode
; 法兰盘加工（车铣复合）

T0101 M03 S2000    ; 车刀
G00 X50.0 Z2.0
Z-5.0
G01 X40.0 F50
X35.0 Z-10.0

G01 X25.0 Z-20.0
G00 X50.0 Z2.0

; 换动力刀
T0202 M06         ; 铣刀，动力刀
G00 X30.0 Z-20.0
G43 Z-15.0 H02    ; 长度补偿
M03 S3000          ; 主轴正转

; 铣键槽
G01 X40.0 Y-5.0 F100
G01 X40.0 Y5.0
G01 X35.0 Y5.0
G01 X35.0 Y-5.0
G00 X50.0 Y0.0
Z2.0

; 铣法兰面
G01 X45.0 Z-20.0 F50
G02 X40.0 Z-25.0 C180.0 F50  ; 铣圆弧，C轴旋转180度
G01 X35.0 Z-30.0
G02 X30.0 Z-35.0 C180.0 F50
G01 X25.0 Z-40.0
G00 X50.0 Z2.0

M05
M30
```

### 6.3 车铣复合编程要点

#### 工艺规划
1. **工序划分**：先车后铣，或边车边铣
2. **装夹方案**：一次装夹完成全部加工
3. **刀具选择**：车刀、铣刀、钻头合理搭配
4. **切削参数**：根据材料优化转速、进给、切深

#### 编程技巧
- **坐标系统一**：确保车削和铣削使用同一坐标系
- **C轴同步**：铣削时C轴随主轴同步旋转
- **避让计算**：避免刀具与工件干涉
- **冷却管理**：车削用冷却液，铣削用切削油

## 八、加工工艺经验

### 8.1 刀具选择原则

| 材料 | 推荐刀具 | 涂层 | 转速 | 进给 |
|------|----------|------|------|------|
| 碳钢 | 硬质合金 | TiAlN | 200-500 | 0.1-0.3 |
| 不锈钢 | 陶瓷/CBN | - | 150-400 | 0.05-0.2 |
| 铝合金 | 金刚石/PCD | - | 500-3000 | 0.2-0.5 |
| 钛合金 | 硬质合金+涂层 | TiAlN | 100-300 | 0.05-0.15 |

### 8.2 切削参数优化

#### 经验公式
```
切削速度 Vc = πDN/1000
; D: 工件直径(mm), N: 主轴转速(rpm)

进给量 f = a_p × n
; a_p: 切深(mm), n: 齿数

切削力 Fc = a_p × a_f × C × d^e
; C, e: 材料系数，d: 刀具直径
```

#### 典型参数表
| 材料 | 切深(mm) | 进给(mm/r) | 转速(rpm) | 切削速度(m/min) |
|------|----------|------------|-----------|------------------|
| 45钢 | 2-5 | 0.1-0.3 | 300-800 | 80-200 |
| 304不锈钢 | 1-3 | 0.05-0.15 | 200-500 | 60-150 |
| 7075铝 | 3-8 | 0.2-0.5 | 800-2000 | 300-800 |
| TC4钛合金 | 1-2 | 0.05-0.1 | 150-400 | 40-100 |

### 8.3 质量检测与补偿

#### 尺寸检测
- **卡尺/千分尺**：常规尺寸检测
- **三坐标测量机**：复杂轮廓检测
- **激光扫描仪**：曲面检测
- **在线检测**：实时监控加工质量

#### 误差补偿
- **热变形补偿**：根据温度变化修正
- **刀具磨损补偿**：定期测量并补偿
- **夹具误差补偿**：修正装夹偏差

## 九、常见问题与解决方案

### 9.1 程序错误

| 错误现象 | 可能原因 | 解决方法 |
|----------|----------|----------|
| 圆弧插补报警 | I/K值错误或符号相反 | 重新计算圆心坐标，检查旋转方向 |
| 刀具补偿报警 | 补偿模式未设置或取消不当 | 确保G41/G42/G40正确使用 |
| 坐标超限 | 程序溢出或计算错误 | 检查程序逻辑，增加保护判断 |
| 主轴失控 | 参数设置错误或传感器故障 | 检查PLC输入输出，测试传感器 |
| 刀库乱码 | 寻址错误或到位信号丢失 | 检查到位开关，重新回参考点 |

### 9.2 加工质量问题

| 缺陷类型 | 产生原因 | 改进措施 |
|----------|----------|----------|
| 尺寸超差 | 刀具磨损、热变形 | 及时换刀，增加检测频次 |
| 表面粗糙 | 转速过低、进给过大 | 提高转速，减小进给 |
| 振纹 | 切削参数不当、刚性不足 | 调整参数，增加支撑 |
| 锥度 | 导轨磨损、丝杠间隙 | 维修导轨，预紧丝杠 |
| 圆度超差 | 主轴跳动、夹具松动 | 修复主轴，紧固夹具 |

### 9.3 系统故障

| 故障现象 | 排查步骤 | 解决方案 |
|----------|----------|----------|
| 系统死机 | 重启系统，检查电源 | 恢复出厂设置，更新固件 |
| 通讯中断 | 检查网线/接口 | 更换线缆，清洁接口 |
| 屏幕无显示 | 检查背光、主板 | 更换显示屏，检修主板 |
| 急停报警 | 检查急停按钮 | 复位按钮，排除机械卡阻 |
| 过热保护 | 检查散热风扇 | 清理灰尘，改善通风 |
