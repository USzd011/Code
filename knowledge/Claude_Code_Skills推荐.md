## 二、详细说明

### 2.1 Superpower - 完整开发流程

**核心能力：**
- TDD（测试驱动开发）
- Code Review（代码审查）
- 自动计划
- 完整开发流程一键搞定

**适用场景：**
- 需要完整开发流程
- 追求代码质量
- 自动化开发

**优势：**
- 装完直接用
- 一键搞定全流程

---

### 2.2 Everything Claude Code - 多角色协作

**核心能力：**
- 子Agent分工
- 多角色协作
- 解决上下文腐化

**适用场景：**
- 复杂项目开发
- 需要多角色协作
- 长期项目维护

**优势：**
- 解决上下文腐化问题
- 子Agent分工明确

**两种选择：**

| **UI UX Pro** | 完整设计系统 | 大型项目、完整设计 |
| **Max/frontend-design** | 轻量设计指导 | 小型项目、快速设计 |

**核心能力：**
- UI设计系统

- 组件设计

**适用场景：**
- 需要UI设计
- 前端开发
- 设计系统搭建

---

### 2.4 sanyuan-skills - 代码审查

**核心能力：**
- SOLID原则审查
- 安全审查
- 性能审查
- 多维度审查

**审查维度：**
1. **SOLID原则** - 代码架构质量
2. **安全审查** - 安全漏洞检测
3. **性能审查** - 性能优化建议

**适用场景：**
- 代码质量把控
- 代码审查
- 重构优化

---
