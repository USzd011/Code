## 第一章 切削速度基础公式与换算

### 1.1 切削速度计算公式

```
Vc = π × D × n / 1000

其中：
- Vc: 切削速度 (m/min)
- D: 工件直径或刀具直径 (mm)
- n: 主轴转速 (rpm)
- π: 圆周率
```

### 1.2 主轴转速计算公式

```
n = 1000 × Vc / (π × D)

反推：已知转速求切削速度
Vc = π × D × n / 1000
```

### 1.3 进给速度计算公式

```
Vf = fz × z × n

其中：
- Vf: 进给速度 (mm/min)
- fz: 每齿进给量 (mm/tooth)
- z: 刀具齿数
- n: 主轴转速 (rpm)
```

### 1.4 切深（背吃刀量）ap

```
ap = (D - d) / 2

其中：
- ap: 切深 (mm)
- D: 工件直径 (mm)
- d: 加工后直径 (mm)
```
