## 一、电路分析与设计

### 1.1 基本电路元件

#### 被动元件
| 元件 | 符号 | 功能 | 单位 | 特性 |
|------|------|------|------|------|
| 电阻 | R | 限制电流 | Ω (欧姆) | V=IR，功率耗散 |
| 电容 | C | 存储电荷 | F (法拉) | Q=CV，隔直通交 |
| 电感 | L | 存储磁能 | H (亨利) | V=Ldi/dt，通直阻交 |
| 变压器 | T | 电压变换 | - | 变压、变流、隔离 |

#### 主动元件
| 元件 | 符号 | 功能 | 应用 |
|------|------|------|------|
| 二极管 | D | 单向导电 | 整流、检波、钳位 |
| 三极管 | Q | 放大/开关 | 放大器、开关电路 |
| MOSFET | M | 功率控制 | 电源、电机驱动 |
| IC | U | 集成功能 | 微处理器、传感器 |

### 1.2 模拟电路基础

#### 运算放大器（Op-Amp）
```gcode
; 理想运放特性
- 开环增益: A → ∞
- 输入阻抗: Zin → ∞
- 输出阻抗: Zout → 0
- 带宽: BW → ∞

; 常用配置
; 1. 反相放大器
;    Vout = -(Rf/Rin) * Vin
;    增益: -Rf/Rin

; 2. 同相放大器
;    Vout = (1 + Rf/Rin) * Vin
;    增益: 1 + Rf/Rin

; 3. 加法器
;    Vout = -(V1*Rf/R1 + V2*Rf/R2)

; 4. 积分器
;    Vout = -1/(RC) ∫Vin dt

; 5. 微分器
;    Vout = -RC dVin/dt
```

#### 滤波器设计
| 类型 | 频率响应 | 应用 | 关键参数 |
|------|----------|------|----------|
| 低通 | 通过低频，衰减高频 | 音频处理 | 截止频率fc=1/(2πRC) |
| 高通 | 通过高频，衰减低频 | 去直流 | fc=1/(2πRC) |
| 带通 | 通过特定频段 | 选频 | Q值=中心频率/带宽 |
| 带阻 | 衰减特定频段 | 陷波 | Q值=中心频率/带宽 |

### 2.2 高速信号处理

#### 传输线效应
- **特征阻抗**：Z0 = √(L/C)
- **延迟**：tpd = √(LC)
- **趋肤效应**：高频电流集中在表面
- **驻波**：阻抗不匹配导致反射

#### 端接技术
| 方式 | 位置 | 优点 | 缺点 |

#### Verilog/VHDL编码示例
```verilog
// 2输入AND门
module and_gate (
    input a,
    input b,
    output y
);
    assign y = a & b;
endmodule

// 4位计数器
module counter (
    input clk,
    input rst,
    output reg [3:0] count
);
    always @(posedge clk or posedge rst) begin
        if (rst)
            count <= 4'b0;
        else
            count <= count + 1;
    end
endmodule

// 状态机 - Moore型
module fsm (
    input clk,
    input rst,
    input start,
    output reg done,
    output reg state_reg
);
    localparam S0 = 2'b00;
    localparam S1 = 2'b01;
    localparam S2 = 2'b10;
    
    reg [1:0] state, next_state;
    
    always @(posedge clk or posedge rst) begin
        if (rst)
            state <= S0;
        else
            state <= next_state;
    end
    
    always @(*) begin
        case (state)
            S0: begin
                if (start)
                    next_state = S1;
                else
                    next_state = S0;
            end
            S1: begin
                if (!done)
                    next_state = S1;
                else
                    next_state = S2;
            end
            S2: begin
                next_state = S0;
            end
            default: next_state = S0;
        endcase
    end
    
    assign done = (state == S2);
    assign state_reg = state;
endmodule
```

## 六、电源设计

### 6.1 DC-DC转换器

#### Buck降压转换器
```
输入Vi → Q(MOSFET) → L → Cout → GND
         ↓
       D(二极管)
         ↓
       Cin(储能)
```
- **效率**：85-95%
- **拓扑**：Buck, Boost, Buck-Boost, Cuk
- **控制方式**：PWM, PFM, PWM+PFM

#### 关键参数计算
```
Vout = D × Vin
Iout = Pout / Vout
Ipeak = Iout + ΔIL/2
ΔIL = (Vin - Vout) × D / (L × fsw)
```

### 6.2 电源完整性

#### 设计要点
- **去耦电容**：就近放置，0.1μF+10μF组合
- **电源平面**：完整，无分割，降低阻抗
- **回流路径**：最短，避免跨分割
- **EMI抑制**：滤波器，屏蔽，接地
- **热设计**：散热片，风道，热仿真
