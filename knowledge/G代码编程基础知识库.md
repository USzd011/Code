## 一、G代码概述

### 1.1 什么是G代码

**定义：**
G代码是数控机床的编程语言，用于控制机床的运动和操作。

**作用：**
- 控制刀具路径
- 设定加工参数
- 控制机床功能
- 实现自动化加工

## 三、程序格式规范

### 3.1 程序结构

```
程序头
├── 程序号（Oxxxx）
├── 安全启动（G90 G80 G40 G49）
└── 设定坐标系

程序体
├── 刀具调用（Txxxx）
├── 主轴启动（M03/M04 Sxxxx）
├── 加工路径（G00/G01/G02/G03）
└── 刀具补偿（G41/G42）

程序尾
├── 刀具退回（G00）
├── 主轴停止（M05）
└── 程序结束（M30）
```

### 3.2 程序示例

```gcode
O0001          ; 程序号
G90 G80 G40 G49 ; 安全启动
G00 X0 Y0 Z50  ; 快速定位到安全位置
T0101 M03 S1000 ; 调用1号刀，主轴正转1000转
G00 X10 Y10    ; 快速移动到加工起点
G01 Z-5 F100   ; 直线插补到加工深度
G01 X50 Y50 F200 ; 直线加工
G00 Z50        ; 退刀
M05            ; 主轴停止
M30            ; 程序结束
```

## 四、关键参数说明

### 4.1 进给速度（F）

**单位：** mm/min（毫米/分钟）
**设定：** F值设定刀具移动速度
**示例：** F100 表示每分钟移动100毫米

### 4.2 主轴转速（S）

**单位：** rpm（转/分钟）
**设定：** S值设定主轴转速
**示例：** S1000 表示主轴每分钟1000转

### 4.3 刀具号（T）

**格式：** Txxxx
**说明：** 前两位为刀具号，后两位为刀补号
**示例：** T0101 表示1号刀，使用1号刀补
