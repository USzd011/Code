## 三、四层架构

### Phase 1：基础层

**包含Agent：**
- A1 PhysicsAgent（物理Agent）
- A2 SupplyAgent（供给Agent）
- A3 DemandAgent（需求Agent）

### Phase 2：分析层

**包含Agent：**
- A4 MarketAgent（市场Agent）
- A5 ValueAgent（价值Agent）
- A6 InvestAgent（投资Agent）

### Phase 3：综合层

**包含Agent：**
- A7 TrendAgent（趋势Agent）
- A8 TheoryAgent（理论Agent）
- A9 ExplorerAgent（探索Agent）

### Phase A：合成层

**包含Agent：**
- ReviewAgent v3（审核Agent）
- 功能：论证质量审核
