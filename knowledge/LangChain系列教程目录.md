## 二、基于SQL实现数据分析问答详解

### 2.1 项目概述

**技术栈：**
- LangChain + Streamlit
- SQLite数据库
- OpenAI API

**核心功能：**
- 从SQLite数据库查询产品类别、品牌或零售商信息
- 自然语言查询转换为SQL
- 智能数据检索和展示

### 2.2 实现方法

**目标：**
```
基于产品类别、品牌或零售商查询从offer_retailer表中提取相关的offer
```

**挑战：**
```
所需数据分散在data目录中的多个表中
采用语言模型(LLM)促进智能数据查询
```

### 2.3 实现步骤

**步骤1：数据库准备**
```
使用存储在data目录中的.csv文件构建本地SQLite数据库
通过sqlite3和pandas库实现
```

**步骤2：LLM集成**
```
通过langchain_experimental.sql.SQLDatabaseChain实现
语言模型(LLM)与本地SQLite数据库的有效交互
```

**步骤3：提示工程**
```
制定合适的提示，指导LLM最优地检索和格式化数据
通过多次迭代和验证来调整这个提示
```

### 2.4 参考资源

**SQLite官网：**
- https://www.sqlite.org/pragma.html

**使用手册：**
- https://www.runoob.com/sqlite/sqlite-select.html
