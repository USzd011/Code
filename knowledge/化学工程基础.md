## 一、化工过程原理

### 1.1 质量守恒定律

#### 连续方程
```
输入量 = 输出量 + 积累量 - 反应消耗量

数学表达式:
dM/dt = Σm_in - Σm_out - R_react × V

其中:
- M: 系统内物质总量 (kg)
- t: 时间 (s)
- m_in: 进入系统的物料流量 (kg/s)
- m_out: 离开系统的物料流量 (kg/s)
- R_react: 反应速率 (kg/(m³·s))
- V: 反应器体积 (m³)
```

#### 能量守恒
```
输入能量 = 输出能量 + 积累能量 + 反应热效应

Q_in = Q_out + dU/dt + ΔH_react × R_react × V

其中:
- Q_in: 输入热量 (J/s)
- Q_out: 输出热量 (J/s)
- U: 系统内能 (J)
- ΔH_react: 反应焓变 (J/kg)
```

### 1.2 动量守恒

#### 伯努利方程（不可压缩流体）
```
P₁/ρg + v₁²/2g + z₁ = P₂/ρg + v₂²/2g + z₂ + h_f

其中:
- P: 压力 (Pa)
- ρ: 密度 (kg/m³)
- g: 重力加速度 (9.81 m/s²)
- v: 流速 (m/s)
- z: 高度 (m)
- h_f: 摩擦损失 (m)
```

#### 动量方程
```
ΣF = d(mv)/dt

对于稳态流动:
ΣF = ṁ(v_out - v_in)

其中:
- F: 作用力 (N)
- m: 质量 (kg)
- v: 速度 (m/s)
- ṁ: 质量流量 (kg/s)
```

### 1.3 传质原理

#### 菲克扩散定律
```
J = -D × dC/dx

其中:
- J: 扩散通量 (mol/(m²·s))
- D: 扩散系数 (m²/s)
- C: 浓度 (mol/m³)
- x: 距离 (m)
```

#### 对流传质方程
```
N_A = k_c(C_A,bulk - C_A,surface)

其中:
- N_A: 组分A的传质速率 (mol/(m²·s))
- k_c: 对流传质系数 (m/s)
- C_A,bulk: 主体浓度 (mol/m³)
- C_A,surface: 界面浓度 (mol/m³)
```

## 二、单元操作

### 2.1 流体输送

#### 管道水力计算
```
压降公式:
ΔP = f × L/D × ρv²/2

其中:
- ΔP: 压降 (Pa)
- f: 摩擦因子（Darcy-Weisbach）
- L: 管长 (m)
- D: 管径 (m)
- ρ: 密度 (kg/m³)
- v: 流速 (m/s)

雷诺数:
Re = ρvD/μ

层流: Re < 2000, f = 64/Re
湍流: Re > 4000, f = 0.316/Re^0.25 (Blasius)
```

#### 泵的选择
```
所需扬程: H_required = ΔP/(ρg) + Δz + h_f

功率计算:
P = ρgQH/η

其中:
- H: 扬程 (m)
- Q: 流量 (m³/s)
- η: 效率 (%)
```

### 2.2 沉降分离

#### 重力沉降
```
终端沉降速度:
v_t = (g(ρ_p - ρ_f)d_p²)/(18μ)

斯托克斯区: Re_p < 1, v_t = g(ρ_p-ρ_f)d_p²/(18μ)
牛顿区: Re_p > 1000, v_t = √(3gd_p(ρ_p-ρ_f)/ρ_f)
过渡区: 1 < Re_p < 1000, 经验公式

处理量:
Q = A × v_t

其中:
- v_t: 终端沉降速度 (m/s)
- ρ_p: 颗粒密度 (kg/m³)
- ρ_f: 流体密度 (kg/m³)
- d_p: 颗粒直径 (m)
- μ: 粘度 (Pa·s)
- A: 沉降面积 (m²)
- Q: 处理量 (m³/s)
```

#### 离心沉降
```
离心分离因数:
K_c = rω²/g

其中:
- r: 旋转半径 (m)
- ω: 角速度 (rad/s)
- g: 重力加速度 (9.81 m/s²)

沉降速度:
v_r = v_t × K_c

其中:
- v_r: 径向沉降速度 (m/s)
- v_t: 终端沉降速度 (m/s)
- K_c: 离心分离因数
```

### 2.3 过滤操作

#### 恒压过滤
```
过滤基本方程:
dV/dt = ΔP/(μ(R_m + R_c))

其中:
- V: 滤液体积 (m³)
- t: 时间 (s)
- ΔP: 压差 (Pa)
- μ: 粘度 (Pa·s)
- R_m: 介质阻力 (1/m)
- R_c: 滤饼阻力 (1/m)

滤饼厚度:
δ = V/A

其中:
- A: 过滤面积 (m²)
- δ: 滤饼厚度 (m)
```
