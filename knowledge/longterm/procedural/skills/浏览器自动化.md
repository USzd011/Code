# 程序知识库 - 浏览器自动化

## 概述

使用 Playwright 进行浏览器自动化操作。

## 核心流程

### 1. 启动浏览器
```
browser action=start
```

### 2. 打开页面
```
browser action=open url=https://example.com
```

### 3. 获取快照
```
browser action=snapshot targetId=<page_id>
```

### 4. 执行操作
```
browser action=act kind=click ref=<element_ref>
browser action=act kind=type ref=<input_ref> text=<content>
```

### 5. 关闭浏览器
```
browser action=stop
```

## 常见问题

### 验证码处理
- 拼图验证码需要人工完成
- 通知用户手动操作
- 完成后继续流程

### 元素定位
- 使用 snapshot 获取 refs
- 优先使用 role refs
- 备选 aria refs

## 最佳实践

1. 先 snapshot 再操作
2. 处理异常情况
3. 等待页面加载
4. 验证操作结果
