# 概念知识库 - OpenClaw

## 定义

OpenClaw 是一个 AI 智能体框架，支持多渠道接入、技能扩展、自主进化。

## 核心特性

### 多渠道支持
- 飞书 (Feishu)
- 企业微信 (WeCom)
- Discord
- Telegram
- Signal
- WhatsApp
- 等 15+ 渠道

### 技能系统
- 540+ 技能包
- 337 个可用技能
- 自动技能检测

### 记忆系统
- 长期记忆 (MEMORY.md)
- 短期记忆 (memory/YYYY-MM-DD.md)
- 类脑知识管理

### 自主性
- 最大自主模式
- 自动执行
- 无需确认

## 技术架构

```
OpenClaw
├── Gateway - 网关服务
├── Skills - 技能系统
├── Memory - 记忆系统
├── Channels - 渠道集成
└── Agents - 智能体框架
```

## 使用场景

- 个人助理
- 自动化工作流
- 内容创作
- 数据处理
- 多智能体协作
