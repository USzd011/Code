> 整理时间：2026-05-04

---

## 一、核心理念

**核心原则：**
```
人做决策，AI做执行
先有故事线，再演示
普通人视角，不是技术文档
```

---

## 二、为什么是4个独立Skill

### 2.1 四大优势

**优势1：独立使用**
```
每个环节可独立使用
已有选题可直接从Step2开始
```

**优势2：人可介入**
```
环节之间人可以介入做判断和调整
```

**优势3：易于优化**
```
单个Skill更小、更容易迭代优化
```

**优势4：不套模板**
```
不套模板，每条视频从信息差出发
```

---

## 三、一条视频的典型流程

### 3.1 完整流程

```
抓选题 → 帮我炼选题 → 确认角度 → 写脚本 → 调整口播稿 → 规划演示 → 开始录制
```

### 3.2 流程详解

**步骤1：抓选题**
```
从信息源抓取选题
筛选5-10条精选
```

**步骤2：帮我炼选题**
```
提炼选题价值
确认内容角度
```

**步骤3：写脚本**
```
根据选题写脚本
调整口播稿
```

**步骤4：规划演示**
```
规划演示内容
开始录制
```

## 四、4个独立Skill

### Skill 1：抓选题

**功能：**
- 从信息源抓取选题
- 筛选精选内容
- 输出选题列表

**使用场景：**
- 每日选题挖掘
- 热点追踪
- 内容规划

### Skill 2：帮我炼选题

**功能：**
- 提炼选题价值
- 确认内容角度
- 输出选题方案

**使用场景：**
- 选题优化
- 角度确认
- 内容定位

### Skill 3：写脚本

**功能：**
- 根据选题写脚本
- 调整口播稿
- 输出完整脚本

**使用场景：**
- 脚本创作
- 口播稿优化
- 内容打磨

### Skill 4：规划演示

**功能：**
- 规划演示内容
- 设计演示流程
- 输出演示方案

**使用场景：**
- 演示规划
- 录制准备
- 内容呈现
