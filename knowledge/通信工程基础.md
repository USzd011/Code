## 一、通信理论基础

### 1.1 信号与系统

#### 连续信号
- **周期信号**：f(t) = f(t+T)
- **非周期信号**：能量有限，功率无限
- **典型信号**：正弦波、方波、三角波、脉冲

#### 频域分析
| 变换 | 公式 | 用途 |
|------|------|------|
| 傅里叶级数 | x(t) = ΣCn·e^(jnω₀t) | 周期信号频谱 |
| 傅里叶变换 | X(ω) = ∫x(t)e^(-jωt)dt | 非周期信号频谱 |
| 拉普拉斯变换 | X(s) = ∫x(t)e^(-st)dt | 系统分析，稳定性 |
| Z变换 | X(z) = Σx(n)z^(-n) | 离散系统，数字信号 |

#### 采样定理
```
奈奎斯特采样定理：fs ≥ 2fmax
- fs: 采样频率
- fmax: 最高频率成分
- 抗混叠滤波器：<fs/2的低通滤波器
```

## 二、无线传播特性

### 2.1 传播模型

#### 自由空间路径损耗
```
Lps(dB) = 20log₁₀(d) + 20log₁₀(f) + 20log₁₀(4π/c)
- d: 距离(km)
- f: 频率(GHz)
- c: 光速(3×10⁸ m/s)
```

#### 常见环境衰减
| 环境 | 衰减因子α | 适用场景 |
|------|-----------|----------|
| 自由空间 | α=2 | 卫星通信 |
| 市区 | α=3-5 | 城市宏站 |
| 郊区 | α=2-3 | 郊区基站 |
| 室内 | α=4-8 | 室内分布 |
| 遮挡 | α=∞ | 视距外 |

### 2.2 多径效应
- **时延扩展**：导致符号间干扰(ISI)
- **相干带宽**：频率选择性衰落
- **相关距离**：空间相关性
- **莱斯衰落**：存在直射路径
- **瑞利衰落**：无直射路径，多径叠加

```
EPC (4G) → SA/NSA (5G) → Cloud Native (5.5G) → AI原生 (6G)
   ↓           ↓              ↓              ↓
  EPC      NGC          CU+DU分离    AI内生
```
