## 一、电力系统分析

### 1.1 电路基本理论

#### 基尔霍夫定律
```
KCL (电流定律): ΣI_in = ΣI_out
    节点处所有流入电流之和等于流出电流之和

KVL (电压定律): ΣV = 0
    回路中所有电压降之和等于电压升之和
```

#### 三相电路
```
对称三相电源:
- 线电压: U_L = √3 × U_phase
- 线电流: I_L = I_phase
- 功率: P = √3 × U_L × I_L × cosφ

连接方式:
- Y型(星形): U_L = √3 × U_phase, I_L = I_phase
- Δ型(三角形): U_L = U_phase, I_L = √3 × I_phase
```

### 1.2 电机原理

#### 直流电机
- **电动势方程**: E = CeΦn
- **转矩方程**: T = CtΦIa
- **转速公式**: n = (U - IaRa)/CeΦ
- **特性**: 硬机械特性，调速方便

#### 异步电机
- **同步转速**: n₀ = 60f/p
- **转差率**: s = (n₀-n)/n₀
- **等效电路**: 转子电阻折算，T型等效
- **机械特性**: 软特性，启动转矩小
- **调速方法**: 变极，变频，变转差率

#### 同步电机
- **功角特性**: T = (3UE₀Xₛ/ω₀)sinδ
- **运行状态**: 发电机，电动机，调相机
- **励磁调节**: V形曲线，无功调节
- **同步调相机**: 改善电网功率因数

### 1.3 变压器

#### 基本原理
- **变压比**: k = N₁/N₂ = U₁/U₂
- **变流比**: I₁/I₂ = N₂/N₁ = k
- **阻抗变换**: Z' = k²Z
- **额定容量**: S_N = U_N × I_N

#### 等效电路
```
一次侧: R₁ + jX₁, 二次侧: R₂' + jX₂'
激磁支路: R_m + jX_m
简化等效: Z_eq = R_k + jX_k
短路损耗: P_k = I_k²R_k
短路电压: U_k = I_kZ_k
```

#### 并联运行条件
- 变比相等，误差<0.5%
- 连接组别相同
- 短路阻抗标幺值相近，偏差<10%
- 容量比不宜过大，一般<3:1
