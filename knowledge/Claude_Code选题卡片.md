## 五、.claude文件夹结构

### 5.1 完整结构

```
.claude/
├── CLAUDE.md      # 员工手册（定义AI身份、风格、约束）
├── memory/        # 工作笔记（自动记录对话、错误、偏好）
├── skills/        # 技能库（预装专业能力、工作流）
```
