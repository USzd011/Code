## 一、项目概述

### 1.1 项目背景
- **工件类型**: 法兰盘零件（材料45钢，直径Φ120mm，厚度20mm）
- **加工要求**: 外圆、端面、螺纹孔M12×1.5
- **设备配置**: FANUC 0i-MF系统，四轴联动
- **刀具配置**: φ80mm可转位刀片，内孔镗刀，螺纹刀
- **加工策略**: 粗加工→半精加工→精加工→攻丝

### 1.2 学习目标
- [ ] 掌握G代码编程规范

- [ ] 具备CAM后处理能力

## 二、基础编程练习

### 2.1 程序结构模板
```gcode
%
O1000                    ; 主程序号
G90 G54 G40 G49        ; 绝对编程，工件坐标系，取消补偿
T1 M06                 ; 换1号刀（φ80mm外圆刀）
S1500 M03              ; 主轴正转1500rpm
G0 X50 Z2             ; 快速定位到起刀点
Z-15                   ; 接近工件
G1 X40 F0.2           ; 直线插补，进给率0.2mm/r
X50 Z-20              ; 倒角
Z-40                   ; 深度
X60                   ; 退刀
G0 Z100               ; 快速抬刀
X100 Y100             ; 返回安全位置
M30                   ; 程序结束
%
```

## 三、固定循环编程

```gcode
%
O4000

X40 Y20            ; 下一孔(40,20)

X20 Y40            ; 下一孔(20,40)
X0 Y40             ; 下一孔(0,40)
G80                ; 取消固定循环
G0 Z100            ; 快速抬刀
X100 Y100          ; 返回安全位置
M30                ; 程序结束
%
```
