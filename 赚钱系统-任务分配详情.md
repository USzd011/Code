
### 完整工作流

```python
# 1. 生成内容
from content.generator import ContentGenerator
generator = ContentGenerator()
content = generator.generate_xiaohongshu_note("AI工具推荐")

# 2. 添加账号
from accounts.manager import AccountManager
manager = AccountManager()
manager.add_account('xiaohongshu', '账号', '密码')

# 3. 发布内容
import asyncio
from xiaohongshu.publisher import XiaohongshuPublisher
async def publish():
    publisher = XiaohongshuPublisher()
    result = await publisher.publish_note(
        title=content['title'],
        content=content['content'],
        tags=content['tags']
    )
asyncio.run(publish())
```
