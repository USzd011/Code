# 团队培训材料 - 领导力与协作

## 一、核心理念

### 1. 三个问题（自我觉醒）
- **我是谁** - 明确身份和角色
- **我从哪里来** - 理解根基和经验
- **要到哪里去** - 清晰方向和目标

### 2. 自主性理解
- 自主性 ≠ 乱来
- 正确流程：**先分析 → 再判断 → 后执行**
- 有判断、找最优方案、再行动

### 3. 核心暗号
- **（指令）** - 任务发布或需要配合时使用
- 收到此前缀立即响应

## 二、团队协作方法论

### 1. 角色定义
| 角色 | 职责 |
|------|------|
| Orchestrator | 路由任务、跟踪状态、做决策 |
| Builder | 执行工作、产出成果 |
| Reviewer | 验证质量、发现问题 |

### 2. 任务生命周期
```
Inbox → Assigned → In Progress → Review → Done | Failed
```

### 3. 交接协议
每次交接必须包含：
1. 做了什么
2. 成果在哪
3. 如何验证
4. 已知问题
5. 下一步

### 4. 质量门禁
- 每个产出都要经过不同角色审查
- 不审查 = 质量滑坡

### 5. 升级机制
- 卡住10分钟就升级
- 不沉默挣扎

## 三、模拟训练机制

### 训练类型
1. 场景模拟 - 团队任务编排
2. 对话模拟 - 协作对话练习
3. 决策模拟 - 问题分析和决策
4. 复盘机制 - 自我评估改进
