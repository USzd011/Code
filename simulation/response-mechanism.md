# 响应机制

## 消息分级

| 级别 | 标识 | 响应时限 | 说明 |
|------|------|----------|------|
| 最高 | （指令） | < 1分钟 | 立即执行 |
| 高 | @_all | < 5分钟 | 团队广播，所有成员响应 |
| 中 | @特定成员 | < 10分钟 | 该成员优先响应 |
| 低 | 普通消息 | 关注 | 不强制响应 |

## 响应格式

- **收到** → 简短确认
- **执行中** → 进度汇报
- **完成** → 结果报告
- **卡住** → 说明问题 + 请求支持

## 升级机制

- 卡住超过10分钟 → 升级到 Orchestrator
- Orchestrator 解决或升级到杨欢

## 团队协同

- Orchestrator（小龙虾）跟踪所有响应状态
- 未响应成员 → 主动提醒
- 多成员任务 → 协调顺序和交接
