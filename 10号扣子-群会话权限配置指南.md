# 10号扣子 - 飞书群会话权限配置指南

> **目标：** 让10号扣子能接收飞书群聊的所有消息，无需@也能响应

---

## 一、扣子平台配置

### 1.1 登录扣子平台

1. 访问：https://www.coze.cn
2. 登录你的账号
3. 找到 **10号扣子** Bot

### 1.2 发布配置

1. 点击 **"发布"** 按钮
2. 选择 **"飞书"** 渠道
3. 进入 **"权限配置"** 页面

### 1.3 开启权限

在权限配置页面，开启以下权限：

| 权限名称 | 权限代码 | 说明 |
|---------|---------|------|
| ✅ 读取群聊消息 | `im:chat:read` | Bot可以读取群聊消息 |
| ✅ 发送群聊消息 | `im:chat:write` | Bot可以发送群聊消息 |
| ✅ 无需@接收消息 | `im:chat:read_without_at` | Bot可以接收所有群消息，无需@ |
| ✅ 获取群组信息 | `im:chat:chat_info:read` | Bot可以获取群组基本信息 |
| ✅ 获取用户信息 | `contact:user.base:read` | Bot可以获取用户基本信息 |

### 1.4 事件订阅

在 **"事件订阅"** 页面，开启以下事件：

| 事件名称 | 事件类型 | 说明 |
|---------|---------|------|
| ✅ 接收消息 | `im.message.receive_v1` | 接收群聊消息事件 |
| ✅ 群组创建 | `im.chat.created_v1` | 群组创建事件 |
| ✅ 群成员增加 | `im.chat.member_added_v1` | 新成员加入群组 |

### 1.5 保存并发布

1. 点击 **"保存"**
2. 点击 **"发布"**
3. 等待发布完成

## 二、飞书开放平台配置（如果需要）

如果扣子平台的配置不生效，需要在飞书开放平台额外配置：

### 2.1 登录飞书开放平台

1. 访问：https://open.feishu.cn
2. 登录你的飞书管理员账号
3. 找到对应的应用（10号扣子）

### 2.2 权限管理

进入 **"权限管理"** → **"API权限"**，开启：

```
✅ im:chat:read - 获取群组消息
✅ im:chat:write - 发送群组消息
✅ im:chat:read_without_at - 不@也能接收消息
✅ im:chat:chat_info:read - 获取群组信息
✅ contact:user.base:read - 获取用户基本信息
```

### 2.3 事件订阅

进入 **"事件订阅"**，添加：

```
✅ im.message.receive_v1 - 接收消息
```

**请求地址：**
```
https://api.coze.cn/webhook/lark/{bot_id}
```

### 2.4 保存配置

点击 **"保存"** 并等待生效

## 三、验证配置

### 3.1 测试步骤

1. 在飞书群聊中发送消息（不@10号扣子）
2. 检查10号扣子是否收到消息
3. 检查10号扣子是否能响应

### 3.2 调试方法

如果配置后仍不生效：

1. **检查Bot状态** - 确认Bot已发布且在线
2. **检查群聊权限** - 确认Bot已加入群聊
3. **检查事件订阅** - 确认事件订阅地址正确
4. **查看日志** - 在扣子平台查看Bot日志

## 四、常见问题

### Q1: 配置后仍收不到消息？

**解决方案：**
1. 确认Bot已发布到飞书
2. 确认Bot已加入群聊
3. 确认权限已生效（可能需要几分钟）
4. 尝试重新发布Bot

### Q2: Bot能收到消息但不响应？

**解决方案：**
1. 检查Bot的触发规则配置
2. 添加关键词触发规则
3. 配置主动参与逻辑

### Q3: 如何让Bot主动参与？

**配置方法：**
1. 在扣子平台配置 **"触发器"**
2. 设置关键词触发规则
3. 配置自动回复逻辑
