# 生活小妙招带货短视频模板

## 📋 模板结构

### 视频时长：30-60秒
- 开头钩子：3-5秒
- 核心内容：20-40秒
- 带货引导：5-10秒

### 画面风格
- 竖屏 9:16
- 明亮温馨色调
- 实拍+字幕叠加
- 快节奏剪辑

---

## 🎬 脚本模板

### 模板一：问题解决型

```
【开头钩子】(3秒)
画面：展示问题场景
字幕：还在为XXX烦恼吗？
音效：疑问音

【核心内容】(25秒)
画面：演示解决方案步骤
字幕：
  1. 第一步...
  2. 第二步...
  3. 第三步...
音效：轻快BGM

【带货引导】(7秒)
画面：展示产品+使用效果对比
字幕：同款神器在橱窗，点击购买！
音效：提示音
```

### 模板二：对比展示型

```
【开头钩子】(3秒)
画面：左右分屏对比
字幕：这样做 vs 那样做
音效：对比音效

【核心内容】(30秒)
画面：错误方法演示 → 正确方法演示
字幕：标注"错误"和"正确"
音效：失败音效+成功音效

【带货引导】(7秒)
画面：产品特写+优惠信息
字幕：限时特惠，点击下方链接
音效：倒计时音效
```

### 模板三：知识科普型

```
【开头钩子】(4秒)
画面：趣味场景
字幕：99%的人都不知道的XXX
音效：惊讶音效

【核心内容】(35秒)
画面：知识点讲解+实操演示
字幕：关键步骤高亮
音效：科普风BGM

【带货引导】(6秒)
画面：产品+二维码
字幕：关注我，获取更多妙招
音效：关注提示音
```
