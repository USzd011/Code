## 🤖 AI生成工作流

### 方案一：全AI生成（适合批量）

```
1. 脚本生成
   → 输入选题到ChatGPT
   → 生成3个版本脚本
   → 选择最优版本

2. AI配音
   → 脚本导入剪映AI配音
   → 选择合适音色
   → 调整语速1.1-1.2倍

3. AI视频生成
   → 脚本分段输入可灵AI
   → 生成对应画面
   → 下载素材

4. 后期合成
   → 剪映导入所有素材
   → 添加字幕、音效
   → 导出成品
```

### 方案二：半AI生成（适合精品）

```
1. 脚本生成
   → AI生成初稿
   → 人工优化润色

2. 真人配音
   → 自己录制或找配音员
   → 情感更自然

3. 实拍+AI混合
   → 核心画面实拍
   → 辅助画面AI生成

4. 精细剪辑
   → 逐帧调整
   → 添加特效
```

## 🔧 自动化脚本（可选）

### Python批量重命名
```python
import os
import re

def batch_rename(folder, prefix):
    files = os.listdir(folder)
    for i, f in enumerate(files):
        ext = os.path.splitext(f)[1]
        new_name = f"{prefix}_{i+1:03d}{ext}"
        os.rename(f"{folder}/{f}", f"{folder}/{new_name}")
```

### FFmpeg批量压缩
```bash
for f in *.mp4; do
  ffmpeg -i "$f" -c:v libx264 -crf 28 -c:a aac "compressed_$f"
done
```
