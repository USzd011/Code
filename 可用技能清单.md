### 3. 小红书发布技能 ✅

**功能：** 自动发布小红书笔记

**使用方法：**
```python
import asyncio
from xiaohongshu.publisher import XiaohongshuPublisher

async def publish():
    publisher = XiaohongshuPublisher()
    result = await publisher.publish_note(
        title="标题",
        content="内容",
        tags=["标签1", "标签2"]
    )
    print(result)

asyncio.run(publish())
```

### 5. 定时发布技能 ✅

**功能：** 定时自动发布内容

**使用方法：**
```python
from scheduler.scheduler import PublishScheduler

scheduler = PublishScheduler()

# 添加定时任务
scheduler.add_task(
    platform='xiaohongshu',
    content={
        'title': '标题',
        'content': '内容',
        'tags': ['标签']
    },
    publish_time='10:00',
    repeat=True
)

# 启动调度器
scheduler.start()
```

**位置：** `/root/.openclaw/workspace/money-system/scheduler/scheduler.py`
