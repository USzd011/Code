## 🔥 待激活升级项（按优先级）

### Step 1 — 开启浏览器沙箱（5分钟）

**现状**: `sandbox.browser.enabled: false` ❌
**操作**: 修改 `openclaw.json` 一行配置

```json
// openclaw.json → agents.defaults.sandbox.browser
{
  "enabled": true    // false → true
}
```

**修改后重启**:
```bash
systemctl restart openclaw-gateway
```

**效果**: 我能自动打开网页、登录网站、操作页面，不用依赖外部操作。

## 📋 一次性执行脚本

如果全部要一次搞定，跑这个：

```bash
# Step 1: 开浏览器沙箱
sed -i 's/"enabled": false/"enabled": true/' /root/.openclaw/openclaw.json

# Step 2: 装核心技能
openclaw skills install agent-browser data-scraper-agent playwright-scraper-skill

# Step 3: 配备用模型
openclaw config set agents.defaults.model.secondary "github/gpt-4.1"

# Step 4: 重启生效
systemctl restart openclaw-gateway
```
